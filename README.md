# Student Score Prediction using Linear Regression

## Project Overview

This project uses a Simple Linear Regression model to predict student scores based on the number of hours studied.

## Dataset

* Source: Kaggle
* Features:

  * **Hours**: Number of hours studied
  * **Scores**: Marks obtained

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Steps Performed

1. Imported required libraries
2. Loaded dataset using Pandas
3. Prepared data (features & target)
4. Trained Linear Regression model
5. Compared Actual vs Predicted values
6. Visualized results using a graph
7. Predicted score for custom input (9.25 hours)

##  Result

* The model successfully predicted student scores
* Example:

  * Study Hours = 9.25
  * Predicted Score ≈ 92.9
